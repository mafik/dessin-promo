Lernen Sie Mind Maps kennen.

Wenn alle Ihre Daten sich so nahe sind, ist die Suche kein Problem mehr.

Mind Maps sind Gedankenlandkarten, in denen sich jedes Element sich so nahe wie möglich bei ähnlichen Elementen befindet. Dies bewirkt, dass es einfacher ist, Sachen, die Sie brauchen, zu finden und schwieriger ist, sie zu vergessen.

Einzigartiges Menü

Dank des Schiebemenüs mit vielen Ebenen können Sie eine belibige Handlung aus einer beliebigen Stelle auf dem Bildschirm starten. Berühren Sie eine beliebige Stelle und schieben Sie mit der Hand auf die Stelle Ihrer Wahl.
